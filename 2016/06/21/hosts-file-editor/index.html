Hosts File Editor

个人作品
  • Qt
  • c/c++
  • hosts

介绍

休息了好久，恢复状态就用 Qt 框架做个小工具吧。经常性的复制编辑 hosts 文件实在有点儿麻烦，索性做个编辑器。Qt 是个好框架，业余零碎的学了点儿皮毛。跨平台、易开发、开源等特点用起来还是不错的，就是最终生成的体积大了点儿。虽然可以自己按需编译 Qt 的源代码不过懒得折腾。

特点概要

  • 默认打开并修改系统默认 Hosts 文件
  • 基于树型列表而非以文本格式解析 Hosts 文件
  • 保存时备份原始文件，自带清空DNS缓存
  • 界面精美，参考QQ音乐设计

    下载地址

    百度云 - 解压密码：rk230

    开发心得

    Qt 的 Model/View/Delegate 模式感觉跟 WPF 的数据绑定都比较经典的 MVC 模式。为了简单用了 Qt 中的 TreeWidget 不需要自己实现 model。程序里用到了继承于 QStyledItemDelegate 的代理类用来处理ip和域名的特殊编辑框。使用中发现 TreeWidget 的 CurrentItem 并没有办法自动清除需要手动 setCurrentItem 赋空值。Qt 中空值可以用 Q_NULLPTR 这个宏。Qt 中的 QMessageBox 竟然没有自带国际化翻译，需要手动从 Qt 的安装路径找到汉化文件 qt_zh_CN.qm 打包到程序中并在代码中使用

Qt Creator 竟然没有图形化的方式设置程序图标。所以麻烦些手动创建资源文件 xxx.rc 加入 IDI_ICON ICON DISCARDABLE "statics/icon/app.ico" 并在 pro 项目文件中 RC_FILE += icon.rc\ 另外因为本程序需要系统文件的读写权限需要新建 uac.manifest 清单文件

Qt 因为有 QStyle 和 QStylesheet 的存在界面可制定化特别给力，几乎跟 css 一样的语法，前端也可以做桌面 UI 不要太爽。参考 QQ 音乐撸了一个感觉还不错。
