VS Code 介绍

发表于 2016-06-16   |   分类于 工具推荐   |     |   阅读次数

前言

写代码写的久了用什么都不是很顺手。大型 IDE 就太笨重有些电脑跑不动，修改个小地方开个项目就要几分钟。普通的编辑器吧又太简陋。处女座的我遭不住了开始如下的一番折腾：

  • notepad++ 一直作为必备工具用了好久，不过毕竟跟不上形式了。
  • Windows 上就别提什么 vim 了没啥原汁原味。
  • Sublime Text 的出现让眼前一亮 - 精美的外观强大的扩展虽然不开源勉强用着还行。
  • Github 的 Atom 仿佛拥有了一切却失去的最基本的速度和流畅性。

    介绍和心得

    好吧这时候微软坐不住了，竟然弄出了 vs code 这个东西竟然还开源你敢信，跟 atom 一样 基于 electron 这个web应用开发内核（研究了下感觉还不错）。优点呢相对于 atom 对 javascript、typescript、markdown、微软系做了强大的原生支持，更好的启动速度，自带 Git 、调试功能、文件比对和分屏。虽然扩展性和界面制定比不上 atom ，关键是启动速度实在是比 atom 好太多！太多！太多！现在更新到1.2版本了，还自带汉化这不是拉拢小白的节奏么。平时开发用 php 和 web 比较多。感觉对 php 的支持不是很好 代码自动完成也只支持基本的函数库并不能多文件，写博文和日记 markdown 用的比较多还是蛮方便的。

推荐几个扩展吧:

  • [background] 设置编辑器背景图自带萌物
  • [Crane - PHP Intellisense] php代码提示
  • [vscode-todo] 代码 TODO 跳转
  • [Material-theme] 各编辑器的经典皮肤
