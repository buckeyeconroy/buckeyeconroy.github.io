Yii2 国际化的使用

发表于 2016-06-27   |   分类于 程序开发   |     |   阅读次数

有关 Yii2 的国际化之前很少使用，今天仔细的研究了下分享下心得

使用框架自带国际化

首先了解到 Yii2 默认有自带已经翻译好了的国际化文件在 vendor/yiisoft/yii2/messages 中。使用只需要在程序的 config 中添加语言配置。

刷新页面会发现诸如表单验证、控件等默认的文字现在已经可以使用中文了。当然框架自带的翻译内容有时候并不完全符合我们自己的使用习惯。

使用自定义国际化文件

我们可以自定义框架默认的国际化的内容。这里不提倡直接修改框架源文件 vendor/yiisoft/yii2/messages 里的内容，那么怎么办呢？可以手工复制目录到自己程序的根目录，不过这里可以用框架自带的命令行工具快速生成 messages 文件。
yii message/config messages/config.php //首先生成国际化的路径和配置文件，路径和文件名建议用默认
会发现已经自动生成相应的文件。打开 config.php 做如下修改

路径默认在 Yii 的源代码中，这里更改为自己项目的 messages 目录中。再次执行命令行
yii message messages/config.php
等待命令结束会自动生成相应的国际化文件，接下来就可以自定义和使用汉化内容了。首先需要修改程序配置文件

代码里只需要使用Yii::t('app', 'hello');//app就是上面配置的前缀名称就可以对文本内容进行国际化。
